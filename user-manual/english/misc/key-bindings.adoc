|===
| Action | MacOS | Windows

| Zoom Centered| kbd:[CMD]+kbd:[M]| kbd:[CTRL]+kbd:[M]

| Zoom to Fit (Max. 100%)| kbd:[ALT]+kbd:[CMD]+kbd:[F]| kbd:[ALT]+kbd:[CTRL]+kbd:[F]
