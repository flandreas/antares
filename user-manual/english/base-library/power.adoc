== Behavior

Emits a single 1 value onto a wire.

The same functionality can be achieved using the more versatile Constant component. The only reason to prefer "Power" is that it is a standard electronic symbol.

== Pins

A power component has only one output pin that constantly emits 1.

== Attributes

Bit Width:: The number of bits of the output pin.
