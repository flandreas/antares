== Behavior

Emits a single 0 value onto a wire.

The same functionality can be achieved using the more versatile Constant component. The only reason to prefer "Ground" is that it is a standard electronic symbol.

== Pins

A ground component has only one output pin that constantly emits 0.

== Properties

Bit Width:: The number of bits of the output pin.
