    Appearance

    Behavior

    A buffer forwards the signal of the input unchanged to the output.

    Table 1. truth table
    In Out

    0

    0

    1

    1

    Z

    0

    X

    X

    (Z: Undefined/High impedance, X: Error)

    The behaviour for unconnected inputs or input wires carrying an undefined signal can be changed using the system preference "Open Gate Input Behaviour".

    The multi-bit version of the buffer will perform its one-bit transformation bitwise on its inputs.

    Mnemonics

    The logical gates of Antares can illustrate their function with so-called "mnemonics". See the chapter Descriptions and explanations. The mnemonic of the buffer is shown below.

    Pins

    Input

    The input to the buffer.

    Output

    The output of the buffer, which outputs the unchanged signal of the input.

    Properties

    Orientation

    The direction in which the output points.

    Bit Width

    The bit width of the signal that can be passed through the buffer.
